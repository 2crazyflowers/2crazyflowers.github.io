University of Minnesota Full Stack Coding Program (November 2017-May 2018)

An intensive and rigorous six month full stack coding program that covered both theory and applications of web development. Curriculum is designed to learn the skills needed to become proficient in both front-end and back-end technologies. The certificate program teaches the knowledge and skills to build dynamic end-to-end web applications using a mix of:
  • lecture to cover the background, history and use of each technology,
  • lab exercises in and out of the classroom to build on the technologies taught,
  • individual assignments graded and commented on in order to get feedback to improve, and
  • group projects to stretch each individuals ability to create full stack web applications the required using both technologies taught in the classroom as well as new technologies not taught to stretch and drive each individual towards greatest learning possible.
